Developer Guide

The documentation is under active development.

Build documentation locally

When sphinx-autobuild is installed, the documentation can bu built locally and automatically updates when one of the source files changes.

sphinx-autobuild.exe doc doc/_build/html

The tabbed code blocks require the package sphinx-tabs.
